# 📊 Bank Marketing Analytics App

Aplicación interactiva desarrollada con **Python** y **Streamlit** para realizar un **Análisis Exploratorio de Datos (EDA)** sobre campañas de marketing bancario.

Este proyecto fue desarrollado como parte de la especialización **Python for Analytics**, aplicando conceptos de análisis de datos, visualización y programación orientada a objetos.

# 🎯 Objetivo del Proyecto

Analizar los factores que influyen en la aceptación de campañas de marketing bancario mediante técnicas de análisis exploratorio de datos (EDA).

La aplicación permite:

- Explorar variables numéricas y categóricas
- Analizar distribuciones
- Detectar valores nulos
- Realizar análisis bivariados
- Generar insights de negocio
- Visualizar información mediante gráficos interactivos
